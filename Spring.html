@EnableAutoConfiguration
attempts to automatically configure your Spring application based on the jar dependencies that you have added.
Example: if HSQLDB is on your classpath, and you have not manually configured any database connection beans, then Spring Boot auto-configures an in-memory database.
Should be applied to @Configuration class.
@SpringBootApplication
It is @SpringBootConfiguration, @EnableAutoConfiguration and @ComponentScan(excludeFilters={...})
